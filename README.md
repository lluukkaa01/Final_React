SkyGlass - ამინდის მონიტორინგის პლატფორმა.  
მრავალგვერდიანი ვებსაიტი, რომელიც შექმნილია React-ისა და Tailwind CSS-ის გამოყენებით.  
  
ძირითადი ფუნქციონალი  
მთავარ გვერდზე ავტომატურად გამოდის 10 წინასწარ გაწერილი პოპულარული ქალაქი (თბილისი, ლონდონი, ნიუ-იორკი და ა.შ.).  
ბრაუზერის გვერდის გადატვირთვის შემდეგ, მთავარი გვერდი ბრუნდება საწყის მდგომარეობაზე (ისევ გამოჩნდება სტანდარტული 10 ქალაქი).  
ძებნის ველის გამოყენებით შესაძლებელია ნებისმიერი ქალაქის დამატება მთავარ პანელზე.  
  
ბარათების მართვა:  
Pin ფუნქციონალი ინახავს ქალაქს რჩეულების სიაში.  
წაშლის ღილაკი მომენტალურად შლის ქალაქის ბარათს მთავარი გვერდიდან.  
თითოეული ქალაქისთვის ხელმისაწვდომია დეტალური მეტეოროლოგიური ანალიზი.   

აპლიკაცია სრულად თარგმნილია ქართულ და ინგლისურ ენებზე.  
ბნელი რეჟიმი: თემებს შორის გადართვა მუშაობს Tailwind class სტრატეგიით.
